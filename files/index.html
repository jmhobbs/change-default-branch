Change Your Default Github Branch

The default branch for git and Github is master. This is an unnecessary and opressive term, and you should change it.

The IETF draft "Terminology, Power and Opressive Language" says:

Master-slave is an oppressive metaphor that will and should never become fully detached from history. Aside from being unprofessional and oppressive it stifles participation...

In addition to being inappropriate and arcane, the master-slave metaphor is both technically and historically inaccurate.

Words have meaning, words have power, and changing them is easy.

If you have many Github repos you would like to update, this tool is for you.
